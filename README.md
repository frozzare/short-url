Short url
==============

[![No Maintenance Intended](http://unmaintained.tech/badge.svg)](http://unmaintained.tech/)

Create a short url that redirects to the long url. Like Simple Address in EPiServer.

![](screenshot.png)

## Install

```
composer require frozzare/short-url
```
